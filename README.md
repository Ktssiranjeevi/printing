# Product Customizer

A React application for customizing products with image upload, color selection, size options, and layer management.

## Features

- Product Info input
- Image upload for customization
- Color picker
- Size selection
- Layer management
- Canvas preview
- Save and Add Product functionality

## Getting Started

### Prerequisites

- Node.js
- npm

### Installation

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:5173](http://localhost:5173) in your browser.

### Build

```bash
npm run build
```

### Lint

```bash
npm run lint
```

## Project Structure

- `src/App.tsx` - Main application component
- `src/App.css` - Styles for the app
- `src/main.tsx` - Entry point
- `src/index.css` - Global styles
- `index.html` - HTML template
- `vite.config.ts` - Vite configuration
- `tsconfig.json` - TypeScript configuration
- `.eslintrc.cjs` - ESLint configuration
